1) What is Cloud Computing?

Ans : Cloud computing means providing services like storage, servers, database, networking, etc.

Ans : Merged Cloud Servers

3) Which of the following are the features of cloud computing?

Ans : Security, Availability, Large Network Access

4) Which of the following is a type of cloud computing service?

Ans : Software-as-a-Service(SaaS)

5) Which of the following is the Cloud Platform provided by Amazon?

Ans : AWS

6) Which architectural layer is used as a backend in cloud computing?

Ans : Cloud

7) Which of the following is the most refined and restrictive cloud service model?

Ans : PaaS(Product-as-a-Service)

Ans : composability

9) Who is the father of cloud computing?

Ans : J.C.R. Licklider

Ans : Desktop as a Service (DaaS)

Ans :  Virtual machines

Ans : A combination of private and public clouds.

Ans : PaaS

14) Which of the following is an example of a public cloud provider?

Ans : Amazon Web Services (AWS)

15) Which of the following is a key concern for organizations when adopting cloud computing?

Ans : Security and privacy

Ans : Cloud migration

Ans : IaaS, PaaS, SaaS

Ans : IaaS
