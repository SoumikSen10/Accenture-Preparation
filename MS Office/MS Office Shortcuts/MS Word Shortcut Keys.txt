New Document:
Ctrl + N

Open Document:
Ctrl + O

Save Document:
Ctrl + S

Save As:
F12 or Alt + F, A

Print Document:
Ctrl + P

Close Document:
Ctrl + W or Alt + F4

Undo:
Ctrl + Z

Redo:
Ctrl + Y

Move One Word to the Right:
Ctrl + Right Arrow

Move One Word to the Left:
Ctrl + Left Arrow

Single-line Spacing:
Ctrl + 1

Double-line Spacing:
Ctrl + 2
